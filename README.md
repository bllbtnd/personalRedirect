# URL Redirector

A simple, secure URL shortener with an admin panel.

Create short links, track clicks, and manage them all in one place at `redirect.ballabotond.com`.

## How It Works

1. **Create**: Go to the admin panel and paste a long URL
2. **Get**: A random 5-character short link is generated
3. **Share**: Send the short link to others
4. **Track**: See how many times your link was clicked
5. **Delete**: Remove links anytime

The admin panel is password-protected. Public redirects work without needing to log in.
